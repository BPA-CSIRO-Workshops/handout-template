Purpose

This repository contains LaTeX style information for use with generating consistently styled handout documents for use in hands-on bioinformatics training workshops.

Example

The example.tex file in this repository is a simple example of how to use the LaTeX style. Have a look at the latest PDFs built from this file to see what the end result is:

  • Trainee's Handout
  • Trainer's Handout

Style Features

  • Consistent styling of text blocks using:
    • Margin icons to guide the readers eye
    • Background shading to increase emphasis of important blocks
  • Code blocks styled for easy copy-and-paste operations
  • Ability to maintain trainer-specific and trainee-specific content in the same source file
  • Trainee-specific styling:
    • Answers to questions hidden
    • Additional vertical space provided following posed questions to allow sufficent room to write answers
  • Trainer-specific styling:
    • Red transperent background on front page
    • "Trainer's Manual" text in footer of all pages
    • Answers to questions displayed in red text

Handout Document Developers

Are you someone looking to develop handout documents for your own hands-on workshop? Would you like to contribute to an existing handout document that was based on the LaTeX style supplied by the handout-template repository? If so, you can find details of how to setup your system, how to use the LaTeX style and how to contribute over on the developers page.
